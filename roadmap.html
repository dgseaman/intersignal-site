Intersignal Roadmap

Last updated: 2025‑08‑10 • Current build: v0.6.1

Vision

Intersignal helps people cut through the fluff. Our stack runs locally when possible, computes narrative integrity diagnostics, and (as Chrome enables it) performs on‑device summaries. Privacy first, zero logins, minimal UI.

Install the Chrome extension from the homepage. Summaries auto‑activate as Chrome exposes window.ai on your device.

Now (v0.6.x)

  • Right‑click action + toolbar popup
  • Integrity Score + Diagnostics + Ontology Tags
  • Status banners for local model availability
  • Clean UI polish and performance pass

Next (R7‑R8)

  • Hybrid summarization fallback (optional): detect missing window.ai and offer local model hookup instructions for power users
  • Context capture improvements: smarter main‑content extraction for heavy SPA/pages
  • Tag tuning: better Thin Sourcing, Strong Title Alignment, hedges/quotes counting

Q4 2025

  • Firefox port (MV3‑compatible)
  • Desktop Dashboard (macOS/Windows) for batch analysis
  • Export: JSON + CSV for research workflows

2026

  • Safari (App Extension) with on‑device summarization when available
  • Pluggable integrity heuristics (research mode)
  • Lightweight team spaces (local‑first, optional sync)

Technical Notes

MV3Client‑sideNo trackingPrivacy‑first

  • Extension remains functional UI‑wise even when the browser hasn’t enabled local summarization yet.
  • As Chrome rolls out the Summarization API, summaries appear without updating the extension.
  • All analysis runs on the user’s device; we don’t store or transmit page content.
