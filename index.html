How It Works

  1. Analyze page text, structure, and media directly in-browser.
  2. Compute ten sub-scores (Readability → Headline + Noise Penalty).
  3. Weight them using your saved rubric → a 0–100 Signal Score.

Install Guide (Chrome Dev / Stable)

  1. Download the ZIP and unzip it.
  2. Open chrome://extensions → toggle Developer mode → click Load unpacked → select the folder.
  3. Pin the extension · Press Alt + S to score · Tune weights under Options.

Privacy

Intersignal Page Scorer sends no page data to servers. Everything is computed locally. JSON exports stay on your device.

FAQ

Why did a news site score higher than mine?

Big outlets usually include dates, bylines, and structured citations — all weighted positively. Increase weights for originality or performance to reflect your focus.

Can I change the rubric?

Yes — open Options and edit each weight. Save and rescore to apply instantly.

What is “Noise” penalty?

A deduction for heavy ad/analytics scripts and iframe bloat. Cleaner sites score higher.
