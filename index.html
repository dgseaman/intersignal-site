How It Works

  1. Analyze page text, structure, and media directly in-browser.
  2. Compute ten sub-scores (Readability → Headline + Noise Penalty).
  3. Weight them using your saved rubric to produce a 0–100 Signal Score.

Install Guide (Chrome)

  1. Download the ZIP and unzip it.
  2. Open chrome://extensions, enable Developer mode, then click Load unpacked and select the folder.
  3. Pin the extension. Press Alt + S to score any page. Adjust weights in Options.

Privacy

Intersignal Page Scorer sends no content to external servers. All computations occur locally. JSON exports are saved only to your device.

FAQ

Why did a news site score higher than mine?

Large outlets often include bylines, timestamps, alt-tagged images, and outbound citations — all weighted positively. Increase originality or performance weights to better match your focus.

Can I change the scoring rubric?

Yes. Open Options in the extension, adjust weights, save, and rescore any page.

What is the “Noise” penalty?

It’s a deduction for ad-heavy pages, tracking scripts, or excessive code weight. Cleaner pages earn higher Signal Scores.
